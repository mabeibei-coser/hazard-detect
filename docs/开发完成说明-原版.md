## 🎯 使用流程

1. **访问应用**: 打开浏览器访问 http://localhost:3001

2. **上传照片**: 
   - 点击"📁 选择照片"选择本地图片
   - 或点击"📷 拍照上传"调用摄像头

3. **选择场景**: 从 10 个预设场景中选择合适的一个

4. **开始识别**: 点击"🔍 开始识别隐患"按钮

5. **查看结果**: AI 分析完成后显示完整报告

6. **导出报告**: 打印或复制报告内容

## 🔧 故障排查

### 1. 前端无法访问
- 检查 3001 端口是否被占用
- 查看浏览器控制台错误信息

### 2. 后端无法启动
- 检查 5001 端口是否被占用
- 查看终端错误日志

### 3. AI 分析失败
- 检查网络连接
- 验证 API Key 是否正确
- 确认图片大小不超过 10MB
