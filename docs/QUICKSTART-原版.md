# 🚀 快速启动指南

## 方法一：一键启动（推荐）

双击 `start.bat` 文件，自动启动前后端服务。

## 方法二：手动启动

### 1. 启动后端（终端 1）

```bash
cd server
npm run dev
```

看到以下输出表示成功：
```
🚀 服务器运行在 http://localhost:5000
```

### 2. 启动前端（终端 2）

```bash
cd client
npm run dev
```

看到以下输出表示成功：
```
VITE ready in XXX ms
➜  Local:   http://localhost:3000/
```

### 3. 访问应用

打开浏览器访问：http://localhost:3000

## 使用流程

1. **上传照片**
   - 点击"📁 选择照片"选择本地图片
   - 或点击"📷 拍照上传"调用摄像头拍照

2. **选择场景**
   - 从 10 个预设场景中选择
   - 默认"⭐ 通用场景"适用于所有环境

3. **开始识别**
   - 点击"🔍 开始识别隐患"
   - 等待 AI 分析（约 5-30 秒）

4. **查看结果**
   - 隐患名称
   - 隐患等级（高/中/低）
   - 具体描述
   - 涉及规范
   - 整改建议
   - 预算经费

5. **导出报告**
   - 点击"📄 打印报告"生成 PDF
   - 点击"📋 复制报告"复制到剪贴板

## 常见问题

### 1. 端口被占用

修改端口：
- 前端：编辑 `client/vite.config.js` 中的 `port`
- 后端：编辑 `server/.env` 中的 `PORT`

### 2. 大模型调用失败

检查：
- 网络连接是否正常
- API Key 是否正确
- 图片大小不超过 10MB

### 3. 摄像头无法使用

- 检查浏览器权限设置
- 确保使用 HTTPS 或 localhost
- 尝试使用文件上传方式
